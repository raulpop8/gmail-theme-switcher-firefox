# Gmail Dark Mode Switcher

This extension automatically toggles dark mode based on system settings for a seamless dark mode experience.

## Features

- 🔄 Sync gmail with system preferences
- 🌙 Instant and Smooth transition
- ⚫️ Inverts colors for dark mode
- 🏞️ Preserves image quality in emails
- 👓 Maintains readability of all content
- ⚡️ Optimised for performance

## Installation

### From Mozilla Web Store

1. Visit the [Mozilla Web Store](https://addons.mozilla.org/en-US/firefox/addon/gmail-dark-mode-switcher/)
2. Click "Add to Firefox"

### Manual Installation (Developer Mode)

1. Download or clone this repository
2. Open Firefox and navigate to `about:debugging`
3. Click on "This Firefox"
4. Click "Load Temporary Add-on..."
5. Select the extension "manifest.json"

## Usage

Once installed, the extension works automatically
